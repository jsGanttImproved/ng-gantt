ng-gantt

Angular wrapper for jsgantt-improved

Options

Live Gantt Chart

Installation

# Install via npm
npm install --save jsgantt-improved ng-gantt

# Import in your Angular module
import { NgGanttEditorModule } from 'ng-gantt';

# Add to your template
<ng-gantt [options]="editorOptions" [data]="data"></ng-gantt>
